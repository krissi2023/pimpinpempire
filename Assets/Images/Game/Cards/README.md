## Card Specifications

- **Format:** PNG with transparency
- **Standard Card Size:** 140x190 pixels (standard playing card ratio)
- **Sprite Sheet:** Organized in 13x4 grid for standard deck
- **Resolution:** High enough for zoom/detail viewing
- **Style:** Consistent with empire aesthetic

## Implementation Notes

- Cards should be designed as sprite sheets for efficient loading
- Maintain consistent sizing and spacing
- Include hover and selected state variations
- Consider animated card effects (flip, glow, etc.)
